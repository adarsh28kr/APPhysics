INCLINE: sin vs cos

"SIN slides ALONG, COS squishes ACROSS"

CONNECTED SYSTEMS — Your 0/6 Topic

Same string = Same tension everywhere  |  Same string = Same |a| for all blocks
Atwood: $a = \frac{(m_2-m_1)g}{m_1+m_2}$    $T = \frac{2m_1 m_2 g}{m_1+m_2}$

FORCE COLOR CODE (Same Everywhere)

Only draw forces that act ON the object. Never draw forces it exerts on others.

N DIRECTION — Always ⊥ to SURFACE

N ≠ mg always! N = mg only on flat surfaces with no vertical forces.
"N is always perpendicular to what you're touching"

FRICTION — Ask This FIRST
