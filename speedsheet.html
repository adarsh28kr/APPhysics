1. KINEMATICS

$v = v_0 + at$
$x = v_0t + \tfrac{1}{2}at^2$
$v^2 = v_0^2 + 2a\Delta x$
$v = \frac{dx}{dt}$, $a = \frac{dv}{dt}$
$\Delta x = \int v\,dt$, $\Delta v = \int a\,dt$
Projectile: $a_x = 0$ always. $a_y = -g$ always. Horizontal & vertical are INDEPENDENT.
See v-t graph → slope = a, area = Δx
See "when does it stop?" → set v(t) = 0, solve for t
See "when does it turn around?" → v = 0 AND a changes sign

2. NEWTON'S LAWS

$\vec{F}_{net} = m\vec{a}$ — apply separately: $\Sigma F_x = ma_x$, $\Sigma F_y = ma_y$
Elevator UP (or accel up): Scale = m(g+a) — feels heavier
Elevator DOWN (or accel down): Scale = m(g−a) — feels lighter
Weightless: a = g (freefall)
Incline (friction): a = g(sinθ − μcosθ)
Constant v on incline: μk = tanθ
Atwood machine: a = (m2−m1)g / (m1+m2)
$F_c = \frac{mv^2}{r}$ toward center
$v_{top} = \sqrt{gR}$ (weightless)
Linear drag: $v_t = mg/b$
Quad drag: $v_t = \sqrt{mg/b}$
TRAP: Linear drag terminal v = mg/b — NO square root! Only quadratic drag has √.

3. WORK, ENERGY & POWER

$KE = \tfrac{1}{2}mv^2$
$U_g = mgh$
$U_s = \tfrac{1}{2}kx^2$
$W = \int \vec{F} \cdot d\vec{r}$
$W_{net} = \Delta KE$
$P = \frac{dW}{dt} = \vec{F} \cdot \vec{v}$
$F(x) = -\frac{dU}{dx}$
Equilibrium: $F = -dU/dx = 0$. Stable if $d^2U/dx^2 > 0$ (valley). Unstable if $< 0$ (hill).
Loop-the-loop min h: h = 5R/2 (set N=0 at top, then energy)
Constant v up hill: P = mgv sinθ (NOT zero!)
Spring launches block: ½kx² = ½mv² + mgh + μmgd
TRAP: Constant velocity ≠ zero power. Engine must overcome gravity/friction.
TRAP: Friction removes energy BOTH ways. Up the incline AND back down = 2× energy loss.

4. MOMENTUM & COLLISIONS

$\vec{p} = m\vec{v}$
$\vec{J} = \int \vec{F}\,dt = \Delta\vec{p}$
Momentum is ALWAYS conserved in collisions. Check KE to classify type.
TypeMomentumKEObjects

Perfectly InelasticConservedMAX lossStick together
1D elastic (target at rest): v1' = (m1−m2)v/(m1+m2), v2' = 2m1v/(m1+m2)
Equal masses elastic: they swap velocities
Explosion from rest: Σp = 0 → m1v1 + m2v2 = 0
Rocket equation: v = ve ln(M0/M)
$x_{cm} = \frac{\sum m_i x_i}{\sum m_i}$    $\vec{v}_{cm} = \frac{\sum m_i \vec{v}_i}{\sum m_i}$
TRAP: "Momentum conserved" does NOT mean elastic. ALWAYS compute KE before & after to check.

6. OSCILLATIONS

$x(t) = A\cos(\omega t + \phi)$
$\omega = \sqrt{k/m}$, $T = 2\pi/\omega$
$v_{max} = A\omega$
$a_{max} = A\omega^2$
$v = \omega\sqrt{A^2 - x^2}$
Pendulum: $T = 2\pi\sqrt{L/g}$
Physical: $T = 2\pi\sqrt{I/(mgd)}$
At x = 0: max speed, all KE, zero PE
At x = ±A: v = 0, all PE, zero KE
KE = 3×PE: at |x| = A/2 (since KE+PE = ½kA²)
Different planet (g changes): T scales as 1/√g. g×4 → T halves.

7. GRAVITATION

$F = \frac{GMm}{r^2}$
$g = \frac{GM}{r^2}$
$v_{orbit} = \sqrt{GM/r}$
$v_{esc} = \sqrt{2GM/r} = \sqrt{2}\,v_{orbit}$
Orbital Energy: $E = -\frac{GMm}{2r}$ < 0    $KE = \frac{GMm}{2r}$    $PE = -\frac{GMm}{r}$
Kepler's 3rd: T² ∝ r³
Speed × √2: escape (E = 0)
Higher orbit: slower speed, longer period, MORE total energy (less negative)
TRAP: Total energy is NEGATIVE for bound orbits. E = −GMm/(2r). More negative = more tightly bound.

8. CROSS-TOPIC PATTERNS — "See This, Do This"

Block slides, enters loop: Energy conservation. At top: N + mg = mv²/R. Min h: set N = 0.
Rolling object up incline: ½mv²(1 + I/mr²) = mgh. Use correct I!
Collision then projectile: Step 1: momentum (find v after). Step 2: projectile (find range/time).
Bullet embeds in block+spring: Step 1: momentum (find v). Step 2: energy (½mv² = ½kx²).
Friction up AND back down: Wfriction = μmg × total distance (both ways!).
Ball thrown tangent to turntable: Angular momentum: L = mvr = Iω
"Is collision elastic?": Compute KEbefore and KEafter. Same? Elastic. Less? Inelastic.
Dimensional analysis: Build answer from given quantities. Check units match.
