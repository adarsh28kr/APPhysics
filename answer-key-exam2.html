Category A: Flat Surface + Forces (Q1-5)
Q1Answer: (A) 1.5 m/s²
Step 1-2 (FBD): Four forces: 60 N right, 20 N left, $f_k = \mu_k mg = 0.25 \times 10 \times 10 = 25$ N left, $N$ up, $mg$ down.
Step 3: Standard axes: $x$ horizontal, $y$ vertical.
Step 4: $F_{net,x} = 60 - 20 - 25 = 15$ N right. $F_{net,y} = 0$.
Step 5: $F = ma$: $15 = 10a \Rightarrow a = 1.5$ m/s²
Teacher's approach: Always compute friction FIRST before anything else. Write $f = \mu mg$ as your very first line. Then list all horizontal forces with signs. Students who rush skip friction and get 4.0 — that's choice B, the trap answer designed for you.
Gotcha: The question has TWO horizontal forces PLUS friction. Three things in the x-equation. If you only subtracted one force from 60, you'd get the wrong answer. Always list ALL forces before writing F=ma.
Common mistakes: (B) 4 m/s² = forgot friction entirely (40/10). (E) 6.5 = subtracted friction from only one applied force. (C) 2.5 = arithmetic error.
Q2Answer: (B) 50 N
Step 1: Check: does the block even move? Max static friction = $\mu_s mg = 0.5 \times 12 \times 10 = 60$ N.
Step 2: Applied force = 50 N. Since $50 < 60$, block does NOT move. Static friction matches exactly: $f_s = 50$ N.
Teacher's approach: This is the #1 friction trap on AP exams. ALWAYS check "does it move?" BEFORE computing acceleration. Step 1: compute max static friction. Step 2: compare to applied force. If applied < max static, the block is stationary and $f_s$ = applied force. If applied > max static, THEN use $\mu_k$ for kinetic friction. Many students skip straight to $f = \mu_k N$ without checking — that's how you get choice (A) = 36 N.
Gotcha: Static friction is NOT always $\mu_s N$. That's the MAXIMUM. If the block isn't on the verge of sliding, $f_s < \mu_s N$. Static friction is whatever it needs to be to prevent motion, up to its maximum.
Common mistakes: (A) 36 N = used kinetic friction $\mu_k mg$ even though block isn't moving. (C) 60 N = used max static friction — that's the max, not the actual value when F < max.
Q3Answer: (A) 0.83 m/s²
Step 3: Standard axes (flat surface → no tilting). Only need x-axis since question asks for horizontal acceleration.
Step 4: Decompose $F_1$: $F_{1x} = 30\cos 60° = 30 \times 0.5 = 15$ N right. $F_2$ is already horizontal: 10 N left.
Step 5-6: $F_{net,x} = 15 - 10 = 5$ N. $a_x = 5/6 = 0.833$ m/s²
Teacher's approach: When a force is at an angle, ALWAYS decompose it BEFORE doing anything else. Write $F_x = F\cos\theta$ and $F_y = F\sin\theta$ as separate lines. Then add ALL x-components. Students who forget to decompose will use the full 30 N and get $a = (30-10)/6 = 3.33$ (choice C) — wrong because only the horizontal COMPONENT matters.
Gotcha: $\cos 60° = 0.5$, NOT $\cos 30°$. Make sure you use the correct angle. The angle is measured from horizontal, so the horizontal component uses cosine of 60°. If you accidentally used $\sin 60° = 0.866$, you'd get $F_{1x} = 26$ N and $a = 2.67$ — not a listed choice, but a common calculator mistake.
Q4Answer: (B) Yes — max friction 30 N < weight 40 N
Step 1-2 (FBD): Four forces: $F = 100$ N pushing RIGHT into wall, $N = 100$ N from wall pushing LEFT, $mg = 40$ N DOWN, friction $f_s$ UP (trying to prevent sliding down).
Step 3: N is perpendicular to the WALL surface → N is horizontal. Friction is along the wall surface → friction is vertical (upward).
Step 4: Max $f_s = \mu_s N = 0.3 \times 100 = 30$ N upward. But weight = 40 N downward. Since 30 < 40, friction can't hold the block.
Step 5: Block slides down. Net downward force = $40 - 30 = 10$ N. $a = 10/4 = 2.5$ m/s² downward (if $m = 4$ kg).
Teacher's approach: Wall problems flip everything students expect. On a floor: N is vertical, friction is horizontal. On a WALL: N is horizontal, friction is vertical. The key question is: can friction (vertical) support the weight? If $\mu_s N < mg$, the block slides. This is a favorite AP conceptual question because it forces you to think about WHICH surface determines N's direction.
Gotcha: Choice (C) says "N exceeds weight so it won't slide." This is the trap for students who confuse N (100 N horizontal) with the force holding the block up (friction, only 30 N). N is horizontal — it pushes the block into the wall, it does NOT hold the block up. Only friction can do that.
Q5Answer: (A) 0.3
Graph analysis: $F = ma + f_k$. The graph is $a$ vs $F$: $a = (F - f_k)/m$. Slope = $1/m = 1/4$, so $m = 4$ kg. x-intercept: $a = 0$ when $F = f_k = 12$ N. $\mu_k = f_k/(mg) = 12/(4 \times 10) = 0.3$.
Category B: Inclines (Q6-12)
Q6Answer: (A) 8 m/s²
Step 1-2: Only $mg$ (straight down) and $N$ (perpendicular to incline). NO friction.
Step 3: Tilt axes: $x$ along incline (down = positive), $y$ perpendicular to incline.
Step 4-5: Along incline: $mg\sin 53° = ma$. Perpendicular: $N = mg\cos 53°$.
Step 6: $a = g\sin 53° = 10 \times 0.8 = 8$ m/s²
Teacher's approach: This is the simplest incline problem — if you can't do this in 15 seconds, you need more practice. The formula $a = g\sin\theta$ should be instant recall for any frictionless incline. Three quick checks: (1) At $\theta = 0°$: $a = 0$ ✓ (flat surface). (2) At $\theta = 90°$: $a = g$ ✓ (free fall). (3) $a$ doesn't depend on mass ✓ (same as free fall).
Gotcha: Choice (B) 6 = $g\cos 53°$. Students mix up sin and cos on inclines. Remember: the component ALONG the incline is $g\sin\theta$ (not cos). Think: at $\theta = 90°$, the "along incline" direction IS straight down, so it should give $g$ — and $\sin 90° = 1$ ✓, $\cos 90° = 0$ ✗.
Q7Answer: (A) μk = 1.0
Step 1: "Constant velocity" is the magic phrase — it means $a = 0$, so net force = 0.
Step 2: Along incline: gravity component down = friction up. $mg\sin 45° = \mu_k mg\cos 45°$.
Step 3: Cancel $mg$ and $\cos 45°$: $\tan 45° = \mu_k = 1.0$.
Teacher's approach: Memorize this golden result: $\mu_k = \tan\theta$ when sliding at constant velocity. It appears on AP exams almost every year in some form. This is also how you experimentally MEASURE $\mu_k$ — tilt a board until the block slides at constant speed, then $\mu_k = \tan\theta$. Also: this is why $\mu_k = \tan\theta_{max}$ gives the maximum angle for static friction. Same idea.
Gotcha: "Constant velocity" does NOT mean the block is stationary. It means it's MOVING but not accelerating. Students sometimes think constant velocity means $v = 0$ — wrong! It means $a = 0$.
Q8Answer: (A) 9 m/s²
Step 1-2 (FBD): Three forces: $mg$ down, $N$ perpendicular to incline, and $F = 20$ N along the incline (downward).
Step 3: Tilt axes. Down the incline = positive $x$.
Step 4-5: Along incline: $mg\sin 30° + F = ma$. $5(10)(0.5) + 20 = 5a$. $25 + 20 = 45 = 5a$. $a = 9$ m/s².
Teacher's approach: The key word is "PUSHED DOWN." Both the gravity component ($mg\sin\theta$) and the applied force point in the SAME direction (down the incline). They ADD. Students who subtract get $a = (25-20)/5 = 1$ — not even a listed choice, so they'd panic. Always ask: "Do these forces point the same way or opposite?"
Gotcha: If the problem said "pushed UP the incline" instead, you'd subtract: $F - mg\sin 30° = ma$ → $a = (20-25)/5 = -1$ m/s² (decelerating, or can't push it up if F < 25). The direction of the applied force completely changes the equation.
Q9Answer: (A) 60 N
System: $a = (F - (m_1+m_2)g\sin 30°)/(m_1+m_2) = (90 - 30)/6 = 10$ m/s²
Isolate $m_2$: $F_{contact} - m_2 g\sin 30° = m_2 a$. $F_{contact} = 4(5 + 10) = 60$ N.
Q10Answer: (A) 0.092 m
Block about to slide UP: $kx = mg\sin\theta + \mu_s mg\cos\theta = 2(10)(0.6 + 0.4 \times 0.8) = 20(0.92) = 18.4$ N
$x = 18.4/200 = 0.092$ m
Q11Answer: (A) 0.60
Key result: Block slides when $\tan\theta_{max} = \mu_s$. $\mu_s = \tan 31° \approx 0.60$.
Q12Answer: (A) 0 N
Key insight: Both blocks have identical $\mu_k$ and same $\theta$. Each independently: $a = g(\sin\theta - \mu_k\cos\theta)$. Mass cancels! SAME acceleration for both.
Result: Since both accelerate identically, the string has no reason to be taut. $T = 0$.
Teacher's approach: This is one of the best conceptual questions on AP exams. Students see "connected by a string" and immediately assume T > 0. But think about it: if you and your friend are both running at the same speed side by side holding a rope, is there tension in the rope? No! Same idea here. Both blocks accelerate at $g(\sin\theta - \mu_k\cos\theta)$ regardless of mass. The string is slack.
Gotcha: This ONLY works because both blocks have the same μ AND the same θ. If one block had different friction, or if they were on different inclines, T would NOT be zero. The mass cancellation is the key insight — on an incline with friction, acceleration is independent of mass (just like free fall!).
Common mistake: Computing $T = (m_2 - m_1)g\sin\theta$ or similar — treating it like an Atwood machine. It's NOT Atwood because both blocks are on the SAME surface, not on opposite sides.

Category D: Circular Motion + Newton (Q19-23)
Q19Answer: (B) $\sqrt{20}$ m/s
Step 1-2: Two forces: $T = 15$ N upward (toward center), $mg = 0.5 \times 10 = 5$ N downward (away from center).
Step 3-5: At BOTTOM, center is UP. Centripetal direction = up. $T - mg = mv^2/r$.
Step 6: $15 - 5 = 0.5v^2/1$. $10 = 0.5v^2$. $v^2 = 20$. $v = \sqrt{20}$ m/s.
Teacher's approach: For vertical circles, always ask: "Where is the center?" At the BOTTOM, center is UP. At the TOP, center is DOWN. Then: forces toward center are positive, forces away from center are negative. Bottom: $T - mg = mv^2/r$ (T toward center, mg away). Top: $mg + T = mv^2/r$ (both toward center). This single framework handles every vertical circle problem.
Gotcha: At the bottom $T > mg$ (you feel heavier — like a roller coaster valley). At the top $T < mg$ (you feel lighter). If you get $T < mg$ at the bottom, your signs are wrong. Quick check: $T = 15 > mg = 5$ ✓.
Q20Answer: (A) $\tan\theta = v^2/(rg)$
$N\sin\theta = mv^2/r$ (horizontal centripetal). $N\cos\theta = mg$ (vertical balance). Divide: $\tan\theta = v^2/(rg)$.
Q21Answer: (A) 0 N (weightless)
Both astronaut and scale are in free fall (orbit IS free fall). The scale has no contact force to measure. Reading = 0.
Q22Answer: (A) $\omega^2 r/g$
Friction provides centripetal force: $\mu_s mg = m\omega^2 r$. $\mu_s = \omega^2 r/g$.
Q23Answer: (A) 3mg
Step 1 — Energy: Released from horizontal means height drop = $L$. $\frac{1}{2}mv^2 = mgL$. So $v^2 = 2gL$.
Step 2 — Newton at bottom: Center is UP. $T - mg = mv^2/L = m(2gL)/L = 2mg$.
Step 3: $T = mg + 2mg = 3mg$.
Teacher's approach: This combines energy AND Newton's Laws — the AP's favorite combination. Always use energy to find speed, THEN use Newton for forces. Never try to do it all with Newton alone (the force changes direction as the pendulum swings). The result $T = 3mg$ at the bottom when released from horizontal is worth memorizing. If released from directly above (180°): $T = 5mg$. General formula: $T = mg(1 + 2\cos\theta_0)$ where $\theta_0$ is the release angle from the bottom... wait actually from the top the formula is $T = mg(3 - 2\cos\theta_0)$ where $\theta_0$ is the initial angle from vertical. At $\theta_0 = 90°$ (horizontal): $T = 3mg$ ✓.
Gotcha: Choice (B) $2mg$ = forgot to add $mg$ back. The centripetal acceleration gives $mv^2/L = 2mg$, but that's only the NET upward force. Tension must ALSO support the weight, so $T = mg + 2mg = 3mg$.
Category E: Calculus-Based Newton (Q24-29)
Q24Answer: (A) 2 m/s
$v = \frac{1}{m}\int_0^{\pi/4} 8\cos(2t)\,dt = \frac{1}{2}[4\sin(2t)]_0^{\pi/4} = \frac{1}{2}(4\sin(\pi/2)) = \frac{1}{2}(4) = 2$ m/s
Q25Answer: (A) $v_0 e^{-bt/m}$
Step 1: Newton's 2nd law: $m\frac{dv}{dt} = -bv$
Step 2: Separate variables: $\frac{dv}{v} = -\frac{b}{m}dt$
Step 3: Integrate both sides: $\ln v - \ln v_0 = -\frac{b}{m}t$. So $\ln(v/v_0) = -bt/m$.
Step 4: Exponentiate: $v = v_0 e^{-bt/m}$
Teacher's approach: This is THE calculus-based Newton's Law problem. It appears on AP Physics C EVERY YEAR in some form. The technique: (1) Write F=ma with the velocity-dependent force. (2) Separate variables (get all v's on one side, all t's on the other). (3) Integrate. (4) Solve for v(t). The result is ALWAYS exponential decay for linear drag. Memorize the form: $v = v_0 e^{-bt/m}$, and know that the time constant $\tau = m/b$ is when the speed drops to $v_0/e \approx 0.37v_0$.
Gotcha: Choice (B) has $e^{-mt/b}$ — the exponent is inverted! Remember: it's $b/m$ in the exponent, not $m/b$. Dimensional check: $[bt/m] = (\text{kg/s})(s)/(\text{kg}) = $ dimensionless ✓. But $[mt/b] = (\text{kg})(s)/(\text{kg/s}) = s^2$ — NOT dimensionless ✗.
Q26Answer: (A) $\sqrt{mg/c}$
Terminal velocity: $a = 0$. $mg = cv_{term}^2$. $v_{term} = \sqrt{mg/c}$.
Q27Answer: (A) $\sqrt{k/m}$
$m\ddot{x} = -kx \Rightarrow \ddot{x} = -(k/m)x$. This is SHM with $\omega^2 = k/m$, so $\omega = \sqrt{k/m}$.
Q28Answer: (A) $Rv$
$F = \frac{dp}{dt} = v\frac{dm}{dt} = vR$ (momentum given to sand each second).
Q29Answer: (A) 4 N
$a(4) = 3 - 0.5(4) = 1$ m/s². $F = ma = 4(1) = 4$ N.
Category F: Advanced / Conceptual (Q30-35)
Q30Answer: (A) Ground pushes horse forward > friction resists cart
The 3rd law pair: Horse pulls cart with force $F$. Cart pulls horse back with force $F$. These are EQUAL and opposite — that's 3rd law.
But the system accelerates because: These are INTERNAL forces (within the horse-cart system). Internal forces cancel when you analyze the system as a whole. What matters are EXTERNAL forces.
External forces: Horse's hooves push backward on ground → ground pushes horse FORWARD (friction). Cart wheels roll on ground → ground provides small rolling friction backward on cart. If horse-ground force > cart-ground friction, the system accelerates forward.
Teacher's approach: This is the most asked conceptual Newton's 3rd law question in all of physics. The key insight: 3rd law pairs act on DIFFERENT objects, so they can never cancel. The horse-on-cart and cart-on-horse forces act on different objects. The horse accelerates because the ground pushes it forward (a DIFFERENT force entirely). Always identify which force acts on which object.
Gotcha: Choices B, C, D, E all contain common misconceptions. (B) "3rd law doesn't apply when accelerating" — WRONG, it always applies. (C) "Horse pulls harder" — WRONG, 3rd law forces are always equal. (D) "Internal forces can accelerate" — WRONG, internal forces cancel for the system. The correct answer is the only one that identifies an EXTERNAL force.
Q31Answer: (A) $\sqrt{gR}$
Weightless means $N = 0$. At top: $mg = mv^2/R$. $v = \sqrt{gR}$.
Q32Answer: (A) Two values (minimum and maximum)
A horizontal force on a steep incline can either (1) push the block into the incline preventing sliding down (minimum F) or (2) push so hard the block slides UP (maximum F). Two equilibrium boundaries exist.
Q33Answer: (A) 2 kg
